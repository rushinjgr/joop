# Installation

## Stable release

To install Justin's Object-Oriented Paradigms, run this command in your terminal:

```
pip install joop
```

This is the preferred method to install Justin's Object-Oriented Paradigms, as it will always install the most recent stable release.

If you don't have [pip](https://pip.pypa.io) installed, this [Python installation guide](http://docs.python-guide.org/en/latest/starting/installation/) can guide you through the process.

## From sources

To install Justin's Object-Oriented Paradigms from sources, run this command in your terminal:

```
pip install git+https://github.com/rushinjgr/joop.git#subdirectory=python
```
